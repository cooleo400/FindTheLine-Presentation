console.log

what is it?

A tool for developers to view the browser’s console.

3Kb js file to include on your page, before any script that has console.log

A tool for developers to view the browser’s console on a browser that does not support console (*cough* IE) or a mobile device.

Shows a viewable console in the browser

How it works

Uses device detection (user agent sniffing) and if console is undefined.

Overwrites the console object and log function.

Adds each log entry as an >li< element into the DOM

Why'd you make it?

See console messages on mobile devices

As well as to see in browsers that do not support console

Limitations/Specifications

Outputing nested object

No libraries

Older browsers dont support JSON.stringify, needed to hardcode it

-- Recursive to print nested objects

Not using any libraries to reduce size and not needed

-- Dom interaction (adding elements, styling) old school

-- Old fashion event handling element.onclick = function(e){}
